Presupuesto de un Periodo

Este es un reporte que permite obtener un informe anual o mensual del presupuesto de un periodo determinado, así como permite también controlar que las adiciones y reclasificaciones presupuestales se hayan realizado en forma correcta, el control que se debe tener en cuenta para verificar que las reclasificaciones presupuestales se hayan realizado correctamente es que el valor anual del presupuesto después de las reclasificaciones no debe cambiar.

Este reporte permite ser exportado a Excel, y realizar las funciones del recuadro.
